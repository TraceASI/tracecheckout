Important notes:
- This is a static GitHub Pages setup.
- Card payments are handled by PayPal's checkout flow, not raw card fields directly hosted by your page.
- The address form is collected before payment and the checkout shows a confirmation section after capture.
- For advanced card fields, webhooks, or server-side verification, you would need a backend.
